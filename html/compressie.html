bitmap en vectoren

Een digitale afbeelding bestaat uit pixels, een afbeelding gemaakt van veel pixels is een bitmap afbeelding. Dit neemt veel opslagruimte in, ook zodra je inzoomt wordt de afbeelding onscherp. Bij een bitmap afbeelding wordt van elke pixel de rgb code voor de pixel opgeslagen dus als een afbeelding van 14 bij 20 pixels hebt dan staat er voor elke pixel een rgb code opgeslagen, dit zijn voor deze afbeelding 14*20=280 rgb pixels met ieder een code die je computer moet opslaan. Dit is veel en je opslag is snel vol. Bij een vector afbeelding worden niet per pixel de code opgeslagen maar worden er vormpjes die in de afbeelding herkent worden van de afbeelding opgeslagen, hier is het zo dat als je inzoomt worden de vormen opnieuw getekend, hierdoor wordt een afbeelding niet onscherp. Bij een vector afbeelding worden de eigenschappen van een afbeelding opgeslagen: de vorm, locatie, grootte en kleur. Dit kan niet bij een digitale foto omdat die niet uit vormen maar uit verschillende kleurtinten bestaat.





geluid

Naast tekst, kleuren en plaatjes staat er ook geluid opgeslagen, geluid zijn golfjes, dit zijn trillingen. Met bijv. een microfoon kunnen geluidstrillingen worden omgezet in elektrische signalen. De computer kan dit signaal omzetten naar digitale waarden, dit heet samplen. Hier wordt een keer in de zoveel tijd de sterkte van het geluid gemeten. Alle punten die gemeten zijn worden samples genoemd, en worden binair opgeslagen. Als het geluid dan afgespeeld wordt dan worden de gemeten waarden weer omgezet naar elektrische signalen. Het aantal samples per seconde wordt de sample rate genoemd.

lossy compressie

Bij lossy compressie worden er stukken van de afbeelding weggelaten, als er veel verschillende tinten kleuren zijn worden vervangen door dezelfde tint zodat het minder ruimte inneemt in de opslag. Dit is niet altijd zichtbaar met het menselijke oog. Een veelvoorkomende lossy compressie methode is jpg.

lossless compressie

Bij lossless compressie worden er geen stukken weggelaten maar wordt er als er dezelfde kleur naast elkaar staat in plaats van een paar keer die code naast elkaar gezet: (hoeveelheid, code). Bij lossless compressie kan het altijd weer terug naar het origineel maar bij lossy compressie kan dit niet. Lossy compressie levert kleinere bestanden op dan lossless compressie

compressie van geluid

De bekendste manier om geluid te compressen is mp3, dit is een vorm van lossy compressie. Bij de kwaliteit van muziek is de sample rate van belang. Ook is het belangrijk hoeveel bits deze sample rate wordt opgeslagen. Deze twee vermenigvuldigd vormt de bitrate van muziek. Dit wordt uitgedrukt in kilobits per seconde (kbps). Mp3 bestanden met een bitrate van 128 kbs dan 1 minuut muziek luisteren kost ongeveer 1 MB aan opslagruimte.
