kleurmodellen

In een computer worden niet alleen letters en getallen gecodeerd maar ook kleuren, als er een foto gemaakt wordt er een momentopname gemaakt van het invallende licht. Een camera bestaat uit meerdere lenzen achter deze lenzen zit een beeldcensor de CCD, charge-coupled-device. Deze sensor bestaat uit miljoenen lichtgevende cellen. De sensoren zetten dit om in een elektrisch singaal.
Een digitale afbeelding bestaat uit pixels, het woord pixel is een combinatie van de woorden picture en element. Pixels zijn vierkante hokjes met een bepaalde kleur.

rgb-model

De pixels zijn zichtbaar als je een foto maakt van een beeldscherm en dan iets inzoomt, de vierkantjes die je dan ziet zijn pixels. Iedere pixel bestaat uit 3 lampjes, door de lampjes feller of zwakker te laten branden kunnen er verschillende kleuren komen. Het rgb model gebruikt de kleuren: rood, groen en blauw (Red, green, blue). Het RGB model is gebasseerd op een zwarte achtergrond. De hoeveelheid van elke kleur die nodig is om de desbetreffende kleur te maken wordt een code van drie getallen gebruikt: (de hoeveelheid rood, de hoeveelheid groen, de hoeveelheid blauw), deze hoeveelheden staan aangegeven met een decimaal getal tussen de 0 en de 255, 0 is bijna niks en 255 is volledig.

CYM-model

Bij een printer wordt niet een zwarte achtergrond gebruikt maar een witte achtergrond want printpapier is wit. Als hier met een rgb code op geprint wordt gaat het mis, hierom bestaat het Cyan-Magenta-Yellow model (cmy-model). Het cmy-model gebruikt de kleuren cyaan, magenta en geel. Om zwart te maken wordt er zwarte inkt toegevoegd omdat alle kleuren mengen donkergrijs geeft. Zo ontstaat het CYMK-model, de k staat voor key of black.
