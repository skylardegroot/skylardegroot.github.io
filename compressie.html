bitmap en vectoren

Een digitale afbeelding bestaat uit pixels een afbeeling gemaakt van veel pixels is een bitmap afbeelding dit neemt veel opslagruimte in, ook zodra je inzoomt wordt de afbeelding onscherp. Bij een bitmap afbeelding wordt van elke pixel de rgb pixel voor pixel opgeslagen dus als een afbeelding van 14 bij 20 pixels hebt dan staat er voor elke pixel een rgb code opgeslagen, dit zijn voor de afbeelding 14*20=280 rgb pixels die je computer moet opslaan. Dit is veel en is je opslag snel vol. Bij een vector afbeelding worden niet de pixels opgeslagen maar worden er vormpjes van de afbeelding opgeslagen, hier is het zo dat als je inzoomt dan worden de vormen opnieuw getekent, hierdoor wordt een afbeelding niet onscherp. Bij een vector afbeelding worden de eigenschappen van een afbeelding opgeslagen: de vorm, locatie, grootte en kleur.
